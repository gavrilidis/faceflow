# FaceFlow — Privacy Policy

**Last updated:** April 2026

## Summary

FaceFlow is a desktop application that runs entirely on your Mac. We do not collect, store, or transmit your photos, face data, or personal information.

---

## Data Processing

### Photos and Face Recognition

All image processing — including face detection, face embedding generation, and person grouping — is performed **locally on your device** using ONNX Runtime. No photos, image previews, face crops, or recognition results are ever sent to external servers.

### License Activation

When you activate FaceFlow with a license key, the following information is sent to our activation server:

- Your license key
- A machine identifier (hardware UUID)

This data is used solely to verify that the license key is valid and has not exceeded its activation limit. No other data is collected during activation.

### AI Model Download

On first launch, FaceFlow downloads two AI model files (~183 MB total) from GitHub. This is a one-time download. After the models are stored locally, no further network requests are made for AI inference.

### Update Checks

FaceFlow periodically checks GitHub Releases for available updates. This check sends only the current app version number. No personal data is included.

---

## Data Storage

All data generated by FaceFlow is stored locally on your Mac:

- **Face embeddings** — stored in a local SQLite database
- **Photo metadata** (ratings, labels, tags) — stored in the same SQLite database
- **Face crop thumbnails** — stored as binary data in SQLite
- **AI models** — stored in the application data directory

No data is synced to the cloud.

---

## Third-Party Services

| Service | Purpose | Data Sent |
|---------|---------|-----------|
| Supabase (activation server) | License key verification | License key + machine ID |
| GitHub Releases | Model download, update checks | App version (for updates only) |

No analytics, telemetry, crash reporting, or advertising SDKs are included in FaceFlow.

---

## Data Retention

FaceFlow does not retain any user data on remote servers beyond the activation record (license key + machine ID). You can request deletion of your activation record by contacting support.

---

## Your Rights

You can:
- Delete all local data by removing the FaceFlow application data folder
- Uninstall FaceFlow at any time without leaving residual data (except for `~/.tauri/faceflow.key` if present)
- Request deletion of your activation record
